Course Catalog Description:

This course will cover the creation of Web pages and sites using HTML, CSS, Javascript and graphical applications. Network multimedia distribution technologies are also explored. The design of effective Web site including page layout, user interface design, graphic design, content flow and asite structure as well as management of Web site resources including internet management and design considerations are addressed. Student design and create a major Web site with multiple pages and cross linked structures. Prerequistes:[none]Credit:2-2-3 Semester Hours

Course Outcomes:

Students completing the course will be able to:

  • Recognize HTML, CSS and Javascript mark-up and code in a web page/application.
  • Select the proper mark-up and code.
  • Indentify improperly used mark-up and code.
  • Produce modern standards compliant web pages.
  • Deploy web pages to the public server.

Course Objectives:

At the conclusion of this course, each successful student will be able to:

  • Explain the client and server architecture of the Internet and related web technologies.
  • Use the basic text editor and other software tools to create web pages using HTML, CSS, Javascript, jQuery, and images.
  • Deploy web pages to a Linux-based web server using sftp.
